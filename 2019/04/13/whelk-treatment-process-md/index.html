重度青春痘治疗过程

文章目录
  1. 1. 药物
  2. 2. 治疗过程
  3. 3. 心态变化
  4. 4. 结果
  5. 5. 小建议

我高三上班学习开始长痘，是爆发性中重度囊肿型痤疮。当时，迫于学习压力、孤身一人以及皮肤健康的历史，我觉得应该很快会消掉，并没有多关注。后续上了大学，正值青春年少，却也没当回事（我内心应该是真心不当回事吧:)）。直达快毕业的时候，偶然和工作室的小伙伴吃饭之际，一个我感觉人很不错的女生，说痘痘一直反复，最近在吃药，希望能彻底根除。与我相比，她的痘痘真是小巫见大巫，我才意识到痘痘对于个人形象带来的负面影响。

我还算白，但由于面部痘痘严重且反复，致使整个面部的色素沉着极其严重，摸着良心说，那应该是张40多岁病态的脸。以前，我零星就医，小到区第一医院，大到浙大附一，但是收效甚浅。长沙痘痘严重的年轻人蛮多的，不像杭州那样干净，水土真的很重要，这大概也成就了湘雅皮肤科的专业性。
下面主要介绍我的使用药物以及治疗过程，

药物

  1. 克拉霉素缓释片：内服消炎
  2. 聚维酮碘溶液、纳米银：外用消炎
  3. 异维A酸：治疗重度囊肿结节痤疮
  4. 维胺颗粒：内服治疗色素沉着
  5. 千百氢醌霜：外用治疗色素沉着

治疗过程

整个治疗过程持续6个月，且上述药物基本上要吃满6个月。其中，克拉霉素缓释片吃了2个月后，停药，主要是我血液中胆固醇以及高密度脂蛋白超标；千百氢醌霜前期效果显著，但是使用两个月后，效果也不明显了，后续经医生推荐使用仙瑟22%高浓度的左旋维生素C(30ml，880一瓶，可真贵啊!)，不过效果是真的顶呱呱，色沉快速消逝。

心态变化

我是个很静的人，但是吃药过程中，特别是异维A酸的服用，导致我莫名心悸、烦躁以及不想做事，这是一个很艰难痛苦的过程，曾经我以为我会抑郁，但是药停后3个月，逐渐好转。这段经历致使我的抗压能力更强了。

结果

基本上，没有爆发性长痘了，皮肤也不大面积出油了。有的时候科研压力大，长几颗痘痘，涂点聚维酮碘溶液消炎即可，很快就消了。

小建议

我一穷二白，平时爱干净，不大懂得男士护肤。因为治痘，逐渐了解和使用了一些护肤品，下面介绍下我认为平价有效的产品：

  1. 洁面：旁氏大米精粹（￥19），氨基酸洗面奶，平价温和
  2. 消炎：聚维酮碘溶液（￥6），痘痘处棉签涂抹放置3分钟
  3. 保湿：百雀羚甘油（￥11），皮肤极其脆弱，甘油极其温和
  4. 祛印：The Ordinary 30% VC(￥100)，左旋VC是医生推荐的，但是仙瑟太贵了，我找到了替代品The Ordinary，真的是价廉物美，但是总体没有仙瑟的惊艳效果，用于日常使用。

中重度青春痘是病，要尽早去三甲级医院治疗，学校的湘雅医院很专业，我平时复诊看了医生给其他不同程度的痤疮患者开的药，基本上都是上述药物。如果三甲医院给你开其他药，你使用后效果不佳，可询问医生开此类药物(考虑到药物不良反应明显，还请慎重！本人只是建议，相应后果自负)。
